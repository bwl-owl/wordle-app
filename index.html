Wordle

How to play

  1. Using either your physical keyboard or the on-screen virtual keyboard, enter a five-letter word attempt and press Enter
  2. The character boxes will change colour according to the below rules:
    3. Character in the correct position:
    X
    Character present in the answer but in the wrong position:
    X
    Character not at all present in the answer:
    X
  3. Using this feedback, try guess the answer before your attempts run out!
